# GUI NordVPN validada

Data: 20260506-150300

O cliente gráfico real da NordVPN foi confirmado como funcional pelo usuário.

## Comandos de validação

```bash
nordvpn status
nordvpn settings
ip route get 1.1.1.1
mocha-nordvpn-gui
mocha-vpn wireguard
mocha-vpn off
```

## Segurança

Não salvar token.
Não salvar cookies.
Não salvar logs brutos com Set-Cookie.
Não rodar instalador externo da Nord no Mocha/NixOS.
